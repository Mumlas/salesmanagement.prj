I. Requirement:
	1. python 3.*
	2. PostgreSQL
	3. Virtual environment - venv, pipenv, etc.
	4. Git repo
	5. Django framework
	
II. Prep
	1. check python is installed
	2. check virtual environment management tool is installed
	3. check PostgreSQL is installed
	4. create Git repo

III. Designing Database
	1. identify tables and their relation and attributes
	2. create a database for the project
	3. create all the identified tables and their attributes
	4. test the database

III. Setting up virtual environment
	1. create a virtual environment
	2. install all the require python packages for the project :Django, psycopg2
	3. test to ensure the packages are installed properly

IV. Create the project
	1. use Django framework to create the project
	2. identify the required apps for the project and create them
	3. create all the folders and subfolders needed for the project
	4. test to ensure the project is created and working properly
